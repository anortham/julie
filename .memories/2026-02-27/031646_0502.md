### What
- **Task 6** (`refresh_stats.rs`): Added `SymbolDatabase` import, query embedding count via `spawn_blocking` before the format block, added `Embeddings: {embed_count}/{symbol_count}` line to stats output
- **Task 7** (`health.rs`): Added conditional embedding line after the existing `push_str` block — shows `Embeddings: N/M (P%)` when embeddings exist, `Embeddings: None` otherwise

### How
- Stats command uses `workspace_db_path(&id)` + `SymbolDatabase::new()` + `embedding_count()` in a `spawn_blocking` to avoid blocking the async runtime
- Health command reads `stats.embedding_count` directly from `DatabaseStats` (populated by `get_stats()` from Task 1)
- Both default to 0 on any error for graceful degradation
