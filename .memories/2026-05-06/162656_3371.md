## Added Markdown semantic symbols

- **WHAT:** Markdown heading symbols now include `heading_level` metadata; inline links, reference definitions, footnotes, and fenced code blocks are extracted as documentation symbols with metadata. Markdown local-heading relationship resolution now ignores non-heading symbols as anchor targets.
- **WHY:** The audit found discarded heading levels, invisible Markdown links/footnotes/code blocks, and a fallback heading text path that could strip too much marker text.
- **HOW:** Added a focused `markdown::semantic_symbols` module under the 500-line file limit, kept heading extraction in `markdown/mod.rs`, and refreshed the Markdown golden fixture for the richer output.
- **IMPACT:** Documentation search and graph data now preserve meaningful Markdown structure instead of flattening it into section text only.
