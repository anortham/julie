## PR #18 fix-batch-2 landed: I5 + I6 + I8 missing-rigor tests + matrix-honesty closure

Second pre-merge fix layer for the tree-sitter extractor audit remediation PR. Closed the remaining material Important items the multi-lane review surfaced (I5/I6/I8) and the dishonesty-gap collateral that I8's meta-test exposed.

### What changed

- **I5** Vue parses-once test (`test_vue_identifier_extraction_parses_script_section_once`) — Form C behavioral assertion: extract a Vue file with a single `doOnce()` call in `<script>`, assert exactly 1 `Call` identifier emitted. Locks down the parses-once invariant Task 12 silently relied on.
- **I6** Data-format relationships exact-target test (`test_data_format_relationships_have_exact_targets_or_no_edge`) — new shared module `crates/julie-extractors/src/tests/data_format_relationships.rs`. YAML alias→anchor and Markdown heading link→heading both produce real edges; non-vacuous helper guards against fixture rot. JSON/TOML documented as not-yet-implemented with capability_gaps pointers.
- **I8 part 1** Two capability-matrix honesty meta-tests added to `tests/capability_matrix.rs`:
  - `test_capability_matrix_pending_claim_requires_pending_output_in_fixtures` — pending_relationships=true must have either a fixture exercising pending or a documented gap.
  - `test_capability_matrix_records_known_gaps_for_languages_with_unfixed_findings` — every false target_capability must have a matching capability_gaps entry.
- **I8 part 2** Closed 19 dishonesty gaps (impl-1's honest-failure rule caught them, impl-2 transcribed lead's curated reasons) across 12 languages: html, css, lua, qml, r, razor, sql, regex, markdown, json, toml, yaml. Reasons are honest about domain limits ("CSS has no static type system") vs. real TODOs ("JSON Schema $ref not yet implemented").

### Why this matters

Tasks 2/12/16 in the audit-remediation plan listed 6 proposed tests that were never written despite being marked Completed. Multi-lane review caught it; this batch locked them in. The matrix dishonesty (12 languages silently lowering the bar) was a pre-existing collateral that I8's meta-test exposed.
