## Embedding Benchmark Baseline + Test Code Waste Discovery

### What
- Built comprehensive embedding quality benchmark (`benchmarks/embedding_quality.py` + `compare_models.py`)
- Ran baseline against 19 reference workspaces (18 languages, 356k symbols) using BGE-small-en-v1.5
- Discovered 42-84% of embedding budget is wasted on test code

### Key Findings
- **Coverage**: ranges from 15.6% (C/jq) to 71% (Rust/Julie) overall; Elixir best at 87% embeddable kinds
- **Quality**: avg top similarity 0.932, avg diversity 0.683, only 9.5% cross-language results
- **Test waste**: Express 84%, Newtonsoft 67%, Julie 66%, Guava 57%, Phoenix 42% test embeddings
- **Variable budget issue**: TS/JS/Python have 30-50% variables, but only 20% budget cap
- **Gap kinds**: Constructor (not embedded, 45-473 per project), Constant, Export

### Benchmark Infrastructure
- `benchmarks/embedding_quality.py` - coverage + KNN similarity analysis per workspace
- `benchmarks/compare_models.py` - side-by-side model comparison from JSON results
- `benchmarks/results/bge-small-en-v1.5.json` - structured baseline data
- 19 workspaces indexed: julie, phoenix, flask, express, zod, guava, Newtonsoft.Json, Slim, sinatra, cats, cobra, jq, nlohmann-json, riverpod, Alamofire, moshi, zls, lite, LabHandbookV2
