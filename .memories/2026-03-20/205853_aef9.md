## Session Complete: Embedding Quality Overhaul

### What was accomplished
Major embedding pipeline improvements committed as `99e7580c`:

1. **Test exclusion** - 47% embedding reduction (83k test embeddings eliminated across 19 workspaces)
2. **Per-language embedding config** - `[embeddings]` section in all 33 language TOML files with `extra_kinds` (Constructor for 11 langs, Export for JS/TS) and `variable_ratio` overrides (0.50 for JS/TS/Ruby/PHP/Lua)
3. **Stale embedding purge** - generalized to clean up any no-longer-eligible embeddings on re-index
4. **Dimension migration** - automatic `recreate_vectors_table` when model dimensions change
5. **Model evaluation infrastructure** - benchmark scripts + baseline results for BGE-small and CodeRankEmbed
6. **Sidecar flexibility** - removed hardcoded 384d, added `trust_remote_code`, `_SUPPORTED_DIMS`
7. **Checklist update** - Check 9 (Embedding Coverage) with baseline table for 18 languages
8. **Elixir verification** - 37/37 checks passing on Phoenix

### Model evaluation findings
- **BGE-small-en-v1.5** (384d, current default): baseline, works everywhere
- **CodeRankEmbed** (768d, nomic-ai): +10% namespace overlap, +20% cross-language, 2.5x slower on MPS. Sidecar-only (no ONNX export). Recommended opt-in via `JULIE_EMBEDDING_SIDECAR_MODEL_ID`
- **Jina-code-v2** (768d): broken on sidecar (transformers 5.x incompatibility), works via ORT but CPU-only on Mac
