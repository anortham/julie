## Razorback Design & Planning Complete

### WHAT
Completed full design and implementation plan for **Razorback** — a Claude Code plugin that forks Superpowers v4.3.1 and adds explicit Julie/Goldfish MCP tool awareness at every exploration and persistence point.

### WHY
Superpowers produces good results but burns tokens and wall-clock time because every agent (controller + subagents) explores codebases with generic Glob/Grep/Read chains. With 9 subagent spawns per 3-task plan, that's ~39-56 exploration rounds. Julie's purpose-built tools can cut that to ~13-24 rounds.

### HOW
- **Faithful fork:** All 14 skills kept with identical process flows
- **Injection points strategy:** Add Julie/Goldfish tool calls at specific places where generic exploration currently happens
- **Highest impact:** 3 subagent prompt templates (implementer, spec-reviewer, code-quality-reviewer) get Julie orientation blocks
- **Session continuity:** Goldfish recall/checkpoint/plan integrated at natural persistence points
- **16 bite-sized implementation tasks** ordered by impact
