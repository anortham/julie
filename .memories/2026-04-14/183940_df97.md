- Cleaned the worktree so only approved Task 1, Task 2, Task 3, and lead-owned baseline files remain changed.

Task 3 existed to stop the extractor crate from having multiple behaviorally distinct public entrypoints. The old API shape let callers bypass canonical parsing and preserved duplicate production routing layers.
