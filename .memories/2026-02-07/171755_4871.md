## Toolset Redesign — Design Complete

### Decision: 14 tools → 9 tools

**Removing 6 tools:**
- `fast_goto` — absorbed into `fast_search` (exact match promotion)
- `edit_lines`, `edit_symbol`, `fuzzy_replace` — host editing tools cover these
- `trace_call_path` — becomes a skill (agent orchestration > heuristic code)
- `fast_explore` — modes become skills

**Adding 1 tool: `deep_dive`**
- Progressive depth: `overview` (default, ~200 tokens) → `context` (~600) → `full` (~1500)
- Kind-aware: functions show callers/callees, traits show implementations, structs show fields/methods, modules show exports
- Reference caps: 10 at overview, 15 at context, uncapped at full
- Replaces the 3-4 tool chain agents currently use to understand a symbol

**Enhancing: `fast_search`**
- Exact symbol name match in definition search → promoted to top with richer definition info

### Implementation Phases
1. **Cut** — remove 6 tools, update registrations + instructions
2. **Enhance** — add exact-match promotion to `fast_search`
3. **Build** — implement `deep_dive` tool
