## Scoped formatter drift before final gates

- **WHAT:** Confirmed the full formatter gate fails on both `origin/main` and the feature branch, then formatted only the eight files whose drift was introduced by this upgrade.
- **WHY:** A whole-repository format rewrite would mix 110 inherited files into the extractor consumer upgrade.
- **HOW:** Compared normalized formatter-drift file lists from an archived `origin/main` tree and the feature worktree. The branch initially added eight drift files; after targeted rustfmt it adds zero. Untouched files followed transitively by rustfmt were restored.
- **IMPACT:** Upgrade-owned Rust is formatter-clean relative to baseline. The full `cargo fmt --check` remains a documented inherited baseline failure, while behavior gates can continue without unrelated churn.
