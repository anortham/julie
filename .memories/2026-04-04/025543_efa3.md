## Edit Tools: Fuzzy Matching + Diff Output Improvements

### Changes since last checkpoint
- **DMP bitap length correction**: `compute_fuzzy_end` uses `diff_main` to walk diff ops and find the true matched content length. Fixes garbled splices when fuzzy-matched content differs in length from old_text
- **Overlapping span filtering**: Trimmed-line matcher now filters overlapping windows (greedy non-overlapping interval selection). Prevents corruption with `occurrence="all"` on repeated content
- **CRLF-aware end position**: End boundary calculation backs up past both `\n` and `\r` for CRLF files. Prevents mixed line endings after replacement
- **Codex review**: GPT-5.4 reviewed the full diff, found both the overlap and CRLF bugs. Both fixed with TDD
