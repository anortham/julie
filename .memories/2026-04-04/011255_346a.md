## Token-Saving Edit Tools: Implementation Complete

### What
Implemented two new DMP-powered MCP editing tools for Julie:

1. **`edit_file`** - Fuzzy find-and-replace for any file. Agent provides old_text/new_text, DMP handles matching. Supports first/last/all occurrence modes. Key discovery: DMP bitap has 32-char limit, so implementation uses exact substring search first with DMP fuzzy fallback.

2. **`edit_symbol`** - Symbol-aware editing via indexed boundaries. Agent references a symbol by name, Julie looks up location via `deep_dive::data::find_symbol`. Operations: replace, insert_after, insert_before.

3. **Markdown section fix** - Expanded section line ranges to cover full content (not just heading). Enables `get_symbols(target="Section Name", mode="minimal")` to return section body content.
