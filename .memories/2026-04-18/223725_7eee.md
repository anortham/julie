## Turned Projects into the dashboard control surface

Implemented the dashboard half of the workspace-management hard-C cut.

- **What changed:** Added Projects actions for `register`, `open`, `refresh`, and `delete`, plus an `Add Workspace` form and a recent cleanup log on the Projects page.
- **UI cleanup:** Removed the dead `Reference Workspaces` detail panel and replaced it with honest workspace state, session-state badges, and row/detail action controls.
- **Lifecycle tracking:** Added current-primary tracking to `SessionTracker` so the dashboard can show `CURRENT`, `ACTIVE`, `KNOWN`, and `MISSING` without guessing.
- **Safety:** Dashboard actions run through `ManageWorkspaceTool` using a disposable temp anchor workspace, then disconnect and clean up the anchor so actions do not leak watcher refs or session counts.

## Why

The dashboard needed to become the real control surface for workspace management after the move away from reference-workspace semantics. Without action routes, cleanup visibility, and truthful state labels, the UI still reflected the old model and could not safely manage known workspaces.

## How

- Added `src/dashboard/routes/projects_actions.rs` and routed POST handlers through `ManageWorkspaceTool`.
- Reworked `src/dashboard/routes/projects.rs` to build one shared Projects view model with session-state badges and cleanup-event rendering.
- Updated Tera templates for the Projects page, table, row, detail, and cleanup log partial.
- Extended `SessionTracker` and handler lifecycle publication to track current primary workspaces per live session.
- Added focused dashboard tests covering page state, add/register, open-without-leak, delete, and detail rendering.

## Impact

- Dashboard users can now add, warm, refresh, and delete workspaces from one page.
- The UI no longer exposes `reference workspace` as a live concept on the Projects surface.
- Current-versus-active state is now real, not inferred from wishful thinking.
- Dashboard regression coverage is stronger, and the full dashboard test slice is green.
