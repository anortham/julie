### Architecture Decisions
- **fastembed cache**: Absolute `~/.cache/fastembed/` path (relative `.fastembed_cache/` breaks in different CWDs)
- **vec0 upsert**: DELETE-then-INSERT (virtual tables don't support INSERT OR REPLACE)
- **Watcher integration**: Added `embedding_provider` to `IncrementalIndexer` instead of changing static handler signatures (22 callers)
- **Graceful degradation**: `embedding_provider` is `Option<Arc<dyn EmbeddingProvider>>` — `None` if model download fails

├── metadata.rs     # Symbol metadata formatting
└── pipeline.rs     # Background + incremental embedding
src/database/
