### Also shipping: xtask runner noise reduction

`xtask/src/runner.rs` + `runner_tests.rs` now capture stdout/stderr and only print it on bucket failure/timeout. Previously every test run dumped its full output into context. These edits were also present in the baseline-export worktree but ship independently of that feature.

### Release flow on tag

`.github/workflows/release.yml` fires on `v*` tags: builds 3 platform archives (win/mac-arm64/linux), creates a GH release with assets, then triggers the `Update Plugin` workflow in `anortham/julie-plugin` which rebuilds the plugin artifact.
