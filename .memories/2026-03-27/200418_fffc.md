## Julie Claude Code Plugin - Design Complete, Plan Written

Designed and planned `anortham/julie-plugin`, a distribution-only Claude Code plugin that packages Julie's pre-built binaries, skills, and behavioral hooks.

### Key Decisions
- **Embed all 3 platform binaries** (~20MB each) rather than download-on-first-run. Plugin cache is version-snapshots so git history bloat is irrelevant.
- **SessionStart hook for behavioral guidance** to compensate for 2k MCP tool description limit. Fires on startup/clear/compact.
- **Surgical tool chains over monolithic calls** in guidance rules. Don't push agents toward deep_dive as default; fast_search -> get_symbols is leaner.
- **Skills copied from julie repo by automation**, not manually maintained. 8 skills ship, search-debug excluded (replaced by dashboard sandbox).
- **Cross-repo GitHub Actions automation**: julie release tags trigger workflow_dispatch in plugin repo.
