## v5.5.5 Released: Day-After Review + CodeRankEmbed Default

### What
Comprehensive review of all 2026-03-21 work (11 commits, 57 files, ~2500 lines). Found and fixed 12 issues across Rust, Python, and docs. Switched macOS/Linux default sidecar model from BGE-small (384d) to CodeRankEmbed (768d). Split oversized test file. Tagged and pushed v5.5.5.

### Critical Fixes
- **UTF-8 panic** in `truncate_on_word_boundary`: sliced by byte index, not char boundary. Any CJK/Cyrillic doc comment would crash the embedding pipeline.
- **OneDrive sync storm**: `default_cache_dir` used `$HOME/.cache/fastembed/` on Windows, landing in the OneDrive-synced user profile. Now uses `LOCALAPPDATA`, `~/Library/Caches`, or `XDG_CACHE_HOME`.

### Important Fixes
- **Stale handle wipe race**: spawned embedding task's self-clear could wipe a newer pipeline's handle. Added `Arc::ptr_eq` guard.
- **WMI AdapterRAM 32-bit overflow**: `uint32` wraps at 4GB. Now tries `nvidia-smi` first, treats WMI < 2GB as wrapped.
- **MPS VRAM**: `driver_allocated_memory()` returns near-zero on fresh process. Switched to `recommended_max_working_set_size()`.
- **GPU batch size on CPU fallback**: GPU-calibrated batch sizes (80+) preserved when falling back to CPU. Now resets to default.
- **Pipeline stats**: `symbols_skipped` was inflated by counting re-enriched symbols. Log now shows new/re-enriched/skipped breakdown.
- Callee name deduplication, penalty stacking docs, CPU probe diagnostic, stale doc comments, `dbg!` added to macro blocklist.

### Model Switch
macOS/Linux default: `BAAI/bge-small-en-v1.5` (384d) → `nomic-ai/CodeRankEmbed` (768d). Pipeline auto-detects dimension mismatch and re-embeds. Benchmarks: +10% namespace, +20% cross-language vs BGE-small.

### Test Split
`embedding_metadata.rs` (1712 lines) → `embedding_metadata.rs` (827) + `embedding_metadata_enrichment.rs` (987). Added 7 new `extract_doc_excerpt` unit tests.

### Review Process
4 parallel code-reviewer agents (embedding pipeline, resolver/handler, sidecar/extractors, tests) + manual deep review of cancellation logic and factory routing. 8/8 dev tier green, 39/39 Python tests green.
