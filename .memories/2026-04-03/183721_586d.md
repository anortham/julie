## Pre-release: v6.5.6 cleanup and fixes

Completed all outstanding items from the embedding enrichment project plus two new fixes:

### Changes
- **fix(deep_dive): accept `symbol_name` alias** -- Some MCP clients send `symbol_name` instead of `symbol`, causing deserialization errors. Added `#[serde(alias = "symbol_name")]`.
- **feat(embeddings): format versioning** -- Added `EMBEDDING_FORMAT_VERSION` constant (v2) and migration 014 (`format_version` column on `embedding_config`). Pipeline detects version mismatch and triggers full re-embed automatically. Users upgrading get enriched embeddings without `force: true`.
- **docs(plugin): update skills and hooks for semantic search** -- Updated SessionStart hook, explore-area, logic-flow, and call-trace skills to mention conceptual/semantic search capabilities.
- **docs: TODO.md cleanup** -- Marked deep_dive alias, format versioning, and plugin distribution as done. Updated NL vocab gap and dogfood notes.
