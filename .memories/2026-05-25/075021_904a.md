## Split Tantivy path-prior tests

### WHAT
Split oversized `src/tests/tools/search/tantivy_path_prior_tests.rs` into focused directory modules under `src/tests/tools/search/tantivy_path_prior_tests/`:
- `query_shape.rs`
- `language_layouts.rs`
- `path_classifiers.rs`
- `scoring_invariants.rs`
- `language_affinity.rs`
- `intent.rs`

### WHY
The original flat test file was 1084 lines, above the project's 1000-line test-file cap and mixed query gating, path classification, scoring constants, language affinity, and test-intent behavior.

### HOW
Kept shared helpers in the directory `mod.rs`, moved tests by behavior category, fixed two initially dropped `#[test]` attributes exposed by `cargo check`, and renamed the test-intent child from `test_intent.rs` to `intent.rs` because the repo/global ignore rules ignore `test_*.rs` child files.

### IMPACT
All 34 original tests are preserved and discovered through the same parent module. Existing `tantivy_` changed-file routing covers the split directory. Cleanup plan and verification ledger now record the slice and fresh Claude review approval.
