## Added Verification Ledger Template And Docs Contract

### WHAT

Created the reusable verification ledger template and documented evidence reuse rules.

### WHY

The architecture cleanup plans need a consistent way to record exact worker gates, lead changed-scope gates, and expensive gate reuse without rerunning slow tests unnecessarily or trusting stale evidence.

### HOW

- Added `docs/plans/verification-ledger-template.md` with required ledger columns and clear reuse rules.
- Updated `docs/TESTING_GUIDE.md` to explain same-HEAD, same-scope evidence reuse.
- Updated `AGENTS.md` to point plans at the ledger template without weakening TDD or worker test ceilings.
- Added docs contract tests to enforce template existence, reuse guidance, and AGENTS linkage.
- Lead review changed the template so the copyable ledger starts empty and example rows live under a separate section marked not to copy.

### IMPACT

Task 4 of the testing-contract plan is complete. Future plan execution can record verification evidence consistently and safely reuse expensive gate results only when HEAD and scope match.
