### Decisions worth preserving
- **PooledConn-wrapping enum over full SymbolDatabase rewrite.** User
  explicitly picked the smaller blast radius. The 280 methods stay
  byte-identical via Deref/DerefMut on `SymbolDatabaseConn`.
- **Drop gating:** SymbolDatabase's WAL-checkpoint Drop runs only for the
  Owned variant. Pooled connections defer checkpoint to the pool owner —
  running it per-request would race other pooled connections on the same
  WAL file.
- **Worker found a real bug while in flight:** `workspace.db_path()`
  diverges from the actual stored path in daemon mode (which uses
  `workspace_db_path(workspace_id)` under the shared indexes dir). Fixed
  by reading `workspace.db.lock().file_path` — the path SymbolDatabase
  actually opened.
- **Daemon test bucket budget bumped** (expected 60s, timeout 180s) in
  `xtask/test_tiers.toml` to accommodate the new A1.2-A1.8 tests.
