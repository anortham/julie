## Codex Review Findings Fixed

Codex (GPT-5.4, xhigh reasoning) reviewed all 8 commits and found 4 issues. Fixed 3 of them; 1 acknowledged as low-risk.

### Fixed

1. **Watcher re-queue busy-spin (High)**: Changed `continue` to `break` after re-queuing, so `tick.tick().await` introduces the actual delay instead of hot-looping in the inner `while let` loop.

2. **Force-index cancels wrong workspace (High)**: `index.rs` was using `handler.workspace_id` (primary) as the cancel key. Now computes the target workspace_id from the path being force-reindexed via `generate_workspace_id`.

3. **edit_symbol/edit_file self-invalidation (Medium)**: After a successful write, both tools now call `db.update_file_hash()` so the freshness guard doesn't reject back-to-back edits on the same file.

### Acknowledged (not fixed)
