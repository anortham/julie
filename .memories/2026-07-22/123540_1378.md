## Stabilized the macOS release toolchain

- **Why:** Homebrew Rust 1.97.1 ships `compiler_builtins` objects marked for macOS 26.0, producing object-version warnings and undermining Julie's macOS 11 support.
- **How:** Drove the change with `toolchain_contract_pins_release_build_inputs`, then rebuilt both release binaries from a clean release profile using the bare repo commands.
- **Impact:** Local `cargo`/`rustc` now resolve to official 1.97.0, release builds are warning-free, and CI cannot drift to a newer stable compiler.
