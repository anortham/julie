## v6.3.0 Release - Dashboard Intelligence Layer

### What
Tagged and releasing v6.3.0 with the complete dashboard intelligence layer:
- New `/intelligence/{workspace_id}` page (fingerprint stats, top symbols by centrality, SVG donut chart for symbol kinds, complexity hotspots, auto-generated story cards)
- Projects page enrichments (top symbol column, kind breakdown bar, intelligence link)
- Metrics page enrichment (tool success rate card, color-coded)
- Search enrichment (centrality badges on top-20 symbols)
- 3 new analytics queries, 1 DaemonDatabase query, 25+ tests
- Bug fixes: template variable mismatch, 404 for inactive workspaces, data files in hotspots, path traversal validation, story card label accuracy, HTML escaping
- Review findings from Gemini and Codex addressed

### Key Design Decisions
- Server-side SVG donut (stroke-dasharray technique, no JS charting lib)
- Direct DB file open for inactive workspaces (bypasses WorkspacePool session side-effects)
- HAVING COUNT > 0 filters data files from hotspots
- Zero schema migrations needed
