## Julie Rescue Phase 1 — `julie-index` crate extracted (relink cure proven)

**WHAT:** Extracted `julie-index` (search + analysis **merged**) one layer above the Phase 0 `julie-core` leaf crate. All `src/search/**` + `src/analysis/**` moved down; top crate re-exports via `pub use` shims (zero call-site edits). 344 decoupled search/analysis tests now live in julie-index's own test binary.

**WHY:** Kill the relink tax — editing a search/analysis test used to relink the 126k-LOC monolith test binary.

**HOW:**
- T1: glob cluster → `julie_core::glob` (severs the one upward edge).
- T2: atomic crate move + shims + `languages/` asset dir; downward repoints (`crate::database`→`julie_core::database`, `crate::extractors`→`julie_extractors`, embeddings-trait→`julie_core::embeddings_contract`).
- T3: relocate 37 test files; ADR-0006 `test-support` feature (gates helper *methods* via feature-flagged dev-dep — thin re-export crate can't re-export methods); minimal pub-promotion (only 4 items).
- T4: dep-direction tripwire + `core-index` xtask bucket/routing/manifest mirror in lockstep.
- T6 (+follow-up): pruned 10 stale `--lib` bucket filters orphaned by the relocation.

**IMPACT:** Relink loop for the 344 relocated tests: **9.46s → 2.95s (3.2×)**, monolith fully insulated (0.53s no-op). dev (35 buckets/1025.8s) + system (7/193.1s) green; 535 test fns preserved exactly (zero dropped); live smoke (search/symbols/deep_dive) green against the new binary.

**KEY LESSON:** `cargo check` never compiles `#[cfg(test)]` — a crate-extraction gate MUST be `cargo nextest run -p julie --no-run`. And test relocation orphans `--lib` bucket filters (nextest exit 4); run a comprehensive 0-match scan over *every* filter after relocating.
