## T2b.6 complete: 48 handler-free tool tests relocated to julie-tools

### WHAT
Relocated every handler-free tool test from `src/tests/tools/` into `crates/julie-tools/src/tests/`. 48 new modules added (on top of the original 5 already there = 53 total). All infra updated to match.

### WHY
Part of Julie Rescue Phase 2b — extracting `julie-tools` as a leaf crate to reduce the relink tax. Tests must live in the crate whose code they exercise so the `-p julie-tools --no-run` compile gate is meaningful.

### HOW

**Relocated groups:**
- Deep dive (2): `deep_dive_regression_tests`, `deep_dive_tests`
- Get context (10): all `get_context_*` except `primary_rebind` and `target_workspace_metrics` (handler-bound)
- Editing (4): `editing_markdown_section`, `editing_security`, `editing_transactional_editing`, `editing_validation`
- Refactoring (3): `refactoring_ast_aware`, `refactoring_compute_line_changes`, `refactoring_import_update`
- Metrics (3 of 4): `metrics_file_size_query`, `metrics_migration`, `metrics_tool_calls_db`
- Search (12): 9 `search_*` + `tantivy_index`, `tantivy_integration`, `tantivy_path_prior`
- Standalone (1): `formatting_tests`

**Key import repointing:** `crate::search::*` → `julie_index::search::*`; `crate::tools::*` → `crate::*` (within julie-tools). One multi-line `use crate::{...}` block in `get_context_task_inputs_tests.rs` escaped sed and was fixed manually.

**Files that STAY (compiler-verified blockers):**
- `metrics/query_tests.rs` — `#[path]` binding to top-crate source; E0433 on `crate::analysis`, `crate::database`, `crate::tools::search`
- All `search/` non-moved — `JulieServerHandler`
- `search/quality.rs` — `path_relevance` + `exact_match_boost` real impls in top-crate `src/utils/`
- `get_symbols_token.rs` — `ContextTruncator` top-crate only
- `editing/edit_file_tests.rs`, both `rewrite_symbol_*` — `JulieServerHandler`
- `refactoring/rename_symbol.rs`, `smart_refactor.rs` — `JulieServerHandler`
- `blast_radius_determinism_tests/**`, `target_workspace_fast_refs_tests/**`, `search/line_mode/**` — `JulieServerHandler` (direct or via `use super::*`)
- `workspace/**`, `search_quality/**` — expected legit stays (handler + workspace helpers)

**Infra updated:** `test_tiers.toml` (13 buckets rewired), `xtask/src/changed.rs` (routing for all new julie-tools test paths), `manifest_contract_expected.rs`, `changed_tests.rs`.
