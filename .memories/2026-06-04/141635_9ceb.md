## T2b.5b complete: julie-tools crate extracted, both gates green

Created `crates/julie-tools` as the handler-free tool crate containing all 9 tool dirs (search, get_context, spillover, impact, deep_dive, navigation, refactoring, editing, symbols). Top crate `src/tools/mod.rs` rewritten as a re-export shim.

### Key changes
- **`crates/julie-tools/`** — new crate with all 9 tool dirs moved in and imports repointed (`crate::X` → crate-local or `julie_core::` / `julie_index::` / etc.)
- **`crates/julie-tools/Cargo.toml`** — deps: julie-core, julie-index, julie-pipeline, julie-context, julie-extractors + tantivy, rusqlite, regex, blake3, uuid, schemars, tree-sitter, diff-match-patch-rs, tokio, serde, serde_json, anyhow, async-trait, tracing. Feature: `test-support = []`
- **`crates/julie-tools/src/lib.rs`** — exports all 9 modules + key type re-exports
- **`src/tools/mod.rs`** — stripped to shim: `pub use julie_tools::X;` for all 9 modules + workspace/metrics stay top-crate
- **`crates/julie-core/src/paths.rs`** — added `WorkspaceFileInputResolution` + `resolve_workspace_file_input` (relocated from top-crate `src/utils/paths.rs` so julie-tools can call it)
- **Bulk `pub(crate)` → `pub`** in all of `crates/julie-tools/src/` (cross-crate re-exports require `pub`)
- **`test-support` feature gate** — 6 files with `#[cfg(test)]` test-helper functions changed to `#[cfg(any(test, feature = "test-support"))]` so top-crate tests can use them: `search/text_search.rs`, `symbols/filtering.rs`, `editing/mod.rs`, `editing/edit_file.rs`, `get_context/pipeline.rs`, `get_context/content.rs`. Complementary `#[cfg(not(test))]` no-ops in `editing/mod.rs` changed to `#[cfg(not(any(test, feature = "test-support")))]`
- **Top-crate `Cargo.toml`** — added `julie-tools` to `[dev-dependencies]` with `features = ["test-support"]`

### sed order-of-operations bug (already fixed)
Step 1 (`crate::tools::search::` → `crate::search::`) ran AFTER step 4 (`crate::search::` → `julie_index::search::`) or vice versa, causing tool-internal search submodule refs to be incorrectly repointed to `julie_index`. Fixed by reverting incorrect substitutions in hint_formatter.rs and execution.rs.
