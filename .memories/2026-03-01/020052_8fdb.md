## Removed Candle Embedding Backend

Deleted the entire Candle embedding provider — 728 lines of implementation code plus ~15 tests and 6 heavy dependencies.

- **WHAT**: Removed `candle_provider.rs`, the `embeddings-candle` feature flag, `Candle` variant from `EmbeddingBackend`, all factory/resolver Candle logic, and all candle-specific tests
- **WHY**: Redundant with both the Python sidecar (primary) and ORT (fallback). Added complexity and 6 heavy deps (`candle-core`, `candle-coreml`, `candle-nn`, `candle-transformers`, `hf-hub`, `tokenizers`) for no unique value
- **HOW**: Systematic removal following the plan — delete provider, update Cargo.toml, clean enum/factory/resolver, remove tests
- **IMPACT**: -2,164 lines, cleaner 2-backend architecture (Sidecar → ORT), faster builds, smaller binary
