## Fixed Standalone CLI Search Readiness

- **WHAT**: Added a regression test for standalone CLI definition search and fixed `bootstrap_standalone_handler` so it runs the workspace index path with embeddings skipped.
- **WHY**: The annotation search dogfood command showed standalone definition search could report `Workspace not indexed yet` after bootstrap, because bootstrap opened workspace handles without filling SQLite and Tantivy.
- **HOW**: Routed standalone bootstrap through `ManageWorkspaceTool` index execution with `skip_embeddings=true`, matching the normal indexing pipeline while keeping CLI startup responsive.
- **IMPACT**: Standalone CLI search can now support autonomous dogfood checks without requiring daemon state or manual restart loops.
