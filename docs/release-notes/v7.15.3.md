# Julie v7.15.3

Patch release that stabilizes the workspace test suite after the recent TODO
audit.

## Test Suite Stability

- Fixed doctest examples for path normalization and string-similarity helpers
  so crate-local documentation compiles under the current workspace layout.
- Marked the reranker scoring formula as text documentation instead of a Rust
  doctest, avoiding a false compile target for prose math.
- Scoped the edit-file test-only before-commit hook to the expected path so
  parallel tests cannot consume another test's hook.
- Lengthened the concurrent MCP watcher observation window to reduce local
  timing flakiness without changing the request concurrency timeout.
- Serialized the HOME-mutating global workspace removal tests on the shared
  `home_env` serial lane.

## Runtime And Compatibility

- No Julie SQLite schema change.
- No extractor dependency change.
- No MCP configuration change.
- Existing workspaces do not require a reindex for this release.
